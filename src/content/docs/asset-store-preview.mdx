This page is a utility preview of the AutoLayout PRO Asset Store marketing suite. It's not part of the public docs flow — it's here so the assets can be reviewed at full resolution before submission to [publisher.unity.com](https://publisher.unity.com).

All images render directly from the source HyperFrames compositions in `auto_layout/launch-video/screenshots/`. Re-render with `./render.sh` after editing.

The display order below matches the intended Asset Store upload order. Filenames preserve their original render index (`01-…`, `02-…`) so the source compositions don't need to be renumbered.
